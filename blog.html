Usage of Relative Size

Percentages is recommended to be used when you want to scale based on the parent property. We use rem with most of properties because it is consistent regardless of their usage. EM should be used while scaling property based on the size of the element.

The difference between Browser pixel and Pixel in CSS

We said 16px is default browser width so what does it mean when we say 200px? How does the browser interpret that?

Pixels are a unit screen measurement. The pixel in browsers show screen resolution and the pixels in CSS show the length of a display area. The pixel (px) in CSS is an angular measurement. It is relative to the resolution of the viewing device. So, the referential pixel is the angle of one pixel on a device with pixel density of 96 dpi and a distance from the reader of an arm’s length (the average length is 28 inches).

Specific Changes in CSS3

1. Selectors: the addition of some new selectors. This selctors allowa you to choose DOM elements without specifying classes and IDs for every element.instead utilize the attribute field to style.
2. Round corners: CSS3 introduces border-radius property to give the round corner effect which use to take a lot of writing code.
3. Border image: in this new version of CSS whith border-image property the border can be changed to an image.
4. Box shadow: this allows you to create a shadow. This effect was achieved by repeated image around the element and now changed to a line code.
5. Text shadow: this property allows you to add a drop shadow to the text. This also used to be done using image or duplicating the text and positioning it a bit lower and aside to give the shadow effect.
6. New colors: this feature added new forms for color and also opacity for transparency. Some are RGBA, HSL, HSLA, Gradient Colors.
7. Web fonts: CSS used to use ‘web-safe’ font and now CSS has allwed to use any font we want by embeding. We define the name, sourse file by using @font-face.
8. Flexbox: this is layout mode designed to enhance object’s alignment, direction and order inside a container. This box model provides block model without using floats.
9. Text overflow: this property deals with the text that is out of the container. There are three options clip will clip the text, ellipsis will present the clipped text with ellipsis” “…””, and initial- it will reset the property to the initial value.
10. Multiple Backgrounds: this property give the website have ,oltiple background images with a simple list according to their appearance.
