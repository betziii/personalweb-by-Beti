The difference between Browser pixel and Pixel in CSS

We said 16px is default browser width so what does it mean when we say 200px? How does the browser interpret that?

Pixels are a unit screen measurement. The pixel in browsers show screen resolution and the pixels in CSS show the length of a display area. The pixel (px) in CSS is an angular measurement. It is relative to the resolution of the viewing device. So, the referential pixel is the angle of one pixel on a device with pixel density of 96 dpi and a distance from the reader of an arm’s length (the average length is 28 inches).

Tan (α) = opposite /adjacent
Tan (α) = ((1/96) / 2) / 28
tan (α) = (1/192) / 28
tan (α) = 1/5376
α = arc tan (1/5376)

Since α is only the half angle we want to find the value of we double the value. So we get:

α = arc tan (1/5376) * 2

In conclusion, we have formulas to convert between radians and pixels.
